# SCT_WD_1

## Project Overview  
SCT_WD_1 is a web development project built during my internship at [SkillCraft Technology] to demonstrate foundational front-end skills.  
It includes HTML, CSS, and JavaScript for a responsive website / web application. 

## Why this project?  
- To practise building responsive layouts and styling with modern CSS.  
- To reinforce JavaScript fundamentals.  
- To create a deployable piece of work that can be showcased on my portfolio.  
- To learn version control and best practices by decoupling assets into folders.

## Built With  
- HTML
- CSS (including media queries for responsiveness)  
- JavaScript 
- Folder structure:  
  - `css/` – Stylesheets  
  - `js/` – JavaScript files  
  - `assets/` – Images, icons, etc.  
  - `index.html` – Entry point  

## Features  
- Responsive navigation bar that adapts for mobile and desktop.  
- Clean, semantic markup and modern CSS layout techniques.  
- Intuitive UI/UX designed to prioritise usability and readability.

## Getting Started  

### Prerequisites  
- A modern web browser (Chrome, Firefox, Edge, Safari)  
- A local development environment (optional, you can simply open `index.html`)
